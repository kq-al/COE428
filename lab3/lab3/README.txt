Insertion Sort: 
	Worst Case - Compares n^2/2: Copies n^2/2: Swaps n^2/2
	Best Case - Compares n-1: Copies 0: Swaps 0
	Average Case - Compares n^2/4: Copies n^2/4: Swaps n^2/4s

Merge Sort: 
	Worst Case - Compares n*log(n): Copies n
	Best Case - Compares 2nlog(n): Copies 2nlog(n)
	Average Case - Compares 2nlong(n): Copies 2nlog(n)
